## Technology Stack

- **Frontend**: React 18.3.1
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI + shadcn/ui
- **Routing**: React Router DOM
- **Form Handling**: React Hook Form + Zod
- **Animations**: Tailwind CSS Animate
- **Icons**: Lucide React
- **State Management**: TanStack Query
